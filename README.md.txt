# Kundli Chatbot 🪐

Kundli Astrology Chatbot using FastAPI & Swiss Ephemeris.

### Features:
- Generate Kundli using birth details
- Shows planetary positions, moon sign, ascendant
- PDF Kundli with chart image
- Built using FastAPI, Swiss Ephemeris, FPDF, Matplotlib

### How to run (locally)
```bash
pip install -r requirements.txt
uvicorn main:app --reload
